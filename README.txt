FAMILY DASHBOARD - SIMPLE USE

1. Open this folder.
2. Double-click Family Dashboard.html.
3. To update the dashboard, replace dashboard-data.js.
4. Refresh the browser.
5. Do not edit app files unless you are changing the dashboard design.

Notes:
- You do not need Command Prompt.
- You do not need localhost.
- You do not need npm.
- ChatGPT should generate the updated dashboard-data.js file.
- The dashboard reads dashboard-data.js when it opens.
