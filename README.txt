FAMILY DASHBOARD - SIMPLE USE

1. Open this folder.
2. Double-click Family Dashboard.html.
3. To update the dashboard, replace dashboard-data.js.
4. Refresh the browser.
5. Do not edit app files unless changing the dashboard design.

Important notes:
- No Command Prompt needed.
- No localhost needed.
- No npm needed.
- ChatGPT should generate dashboard-data.js.
- The dashboard reads dashboard-data.js when it opens.
- Family Dashboard.html is the only launch file.
- dashboard-data.js is the only file you replace for updated family information.
- dashboard-data.js must define window.DASHBOARD_DATA = { ... }.

What each app file does:
- Family Dashboard.html opens the dashboard and loads dashboard-data.js before app.js.
- dashboard-data.js contains the family dashboard information.
- app.js adapts dashboard-data.js into the visual Family Desk dashboard.
- styles.css contains the dashboard design.
